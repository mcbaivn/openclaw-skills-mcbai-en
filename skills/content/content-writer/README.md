# ✍️ Content Writer - MCB AI

> Viết social media posts chuyên nghiệp từ bài nghiên cứu và nguồn dữ liệu. Hỗ trợ LinkedIn, Facebook, Twitter/X, TikTok, Threads — 6 format, 8 tone, EN/VI.

---

## Cài đặt

```bash
npx clawhub@latest install content-writer-mcbai
```

---

## Tính năng

| Tính năng | Chi tiết |
|-----------|---------|
| 📱 Đa nền tảng | LinkedIn, Facebook, Twitter/X, TikTok caption, Threads |
| 📋 6 Format | Toplist, POV, Case Study, How-to, Story, Hook-List-CTA |
| 🎭 8 Tone | Default, Bold, Educational, Storytelling, Analytical, Viral, Empathetic, Custom |
| 📏 3 Độ dài | Short / Medium / Long |
| 🌐 2 Ngôn ngữ | Tiếng Việt / English |
| 🔗 Tích hợp | Nhận nguồn bài trực tiếp từ `content-research` |

---

## Cách dùng

```
Viết LinkedIn post từ bài này: [URL hoặc nội dung bài]
Viết Facebook post dạng Story, tone Storytelling, tiếng Việt
Viết 3 variants cho Twitter/X từ dữ liệu này
```

---

## Formats

| Format | Dùng cho |
|--------|---------|
| 📊 Toplist | Danh sách có số liệu |
| 💡 POV | Quan điểm táo bạo dựa trên data |
| 🔬 Case Study | Phân tích sâu một câu chuyện |
| 🗺️ How-to | Hướng dẫn từng bước |
| 📖 Story | Kể chuyện cảm xúc (Facebook-optimized) |
| 🪝 Hook-List-CTA | Viral format cho Facebook |

---

## Pipeline hoàn chỉnh

```
content-research → chọn bài → content-writer → post LinkedIn/Facebook
```
